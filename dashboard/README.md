# ☕ Coffee Vending Machine | Business Dashboard

Разведочный анализ данных и визуальный бизнес-дашборд кофейного автомата за **2024–2025 год**. Ноутбук превращает сырые транзакционные логи в конкретные инсайты: динамика выручки, популярность напитков, поведение клиентов и паттерны пиковой нагрузки.

## Зачем это важно

Кофейные автоматы генерируют непрерывный поток поведенческих данных, который владельцы почти никогда не используют дальше простого подсчёта выручки. Этот проект показывает, что даже один автомат содержит достаточно сигнала для принятия конкретных бизнес-решений:

- **Оптимизация выручки** - анализ месячного тренда выявляет периоды роста и спада, позволяя точно планировать акции и пополнение запасов вместо интуитивных догадок.
- **Прогнозирование спроса** - тепловая карта час × день недели показывает реальную пиковую нагрузку, что напрямую влияет на расписание обслуживания, закупку ингредиентов и возможное расширение парка автоматов.
- **Оптимизация меню** - количество заказов и выручка по каждому напитку показывают, что держит бизнес, а что занимает слот без отдачи.
- **Удержание клиентов** - анализ интервалов между визитами для держателей карт даёт эмпирическую основу для программ лояльности и выявления оттока.
- **Планирование платёжной инфраструктуры** - соотношение наличных и карты в динамике помогает обосновать приоритет перехода на безнал.

Более широкое значение - методологическое: этот пайплайн является переиспользуемым шаблоном. Любой оператор со схожим автоматом и CSV-выгрузкой может подставить свои данные и получить тот же дашборд за несколько минут.

## Датасет

Источник: [Kaggle: Coffee Vending Machine Transactions](https://www.kaggle.com/datasets/ihelon/coffee-sales)

Два CSV-файла за разные периоды объединяются при загрузке:

| Файл | Период |
|------|--------|
| `index_1.csv` | 2024 |
| `index_2.csv` | 2025 |

Используемые столбцы: `datetime`, `date`, `coffee_name`, `money`, `cash_type`, `card`.

Файлы данных не входят в репозиторий, скачай с Kaggle и положи в `data/`.

## Структура ноутбука

| Секция | Описание |
|--------|----------|
| **0 — Настройка** | Кофейная тёмная палитра, глобальные стили `matplotlib` / `seaborn` |
| **1 — Загрузка данных** | Объединение CSV, парсинг дат, инженерия признаков `hour / weekday / month / week / year` |
| **2 — KPI-плитки** | Общая выручка, количество транзакций, средний чек, доля карт, топ-напиток, пиковый час |
| **3 — Динамика выручки** | Месячные тренды выручки и количества транзакций |
| **4 — Популярность напитков** | Горизонтальные бар-чарты - заказы и выручка по напиткам |
| **5 — Тепловая карта** | Интенсивность продаж по часу и дню недели |
| **6 — Способ оплаты** | Пайчарт наличные vs карта; динамика по месяцам |
| **7 — Топ клиенты** | Топ-15 держателей карт по тратам; любимый напиток каждого |
| **8 — Средний чек** | По напиткам и по дням недели; выделение наиболее маржинальных комбинаций |
| **9 — Retention** | Распределение интервалов между визитами; топ-20 самых частых покупателей |

## Визуальный стиль

Дашборд использует кастомную палитру **espresso-dark**:

| Роль | HEX | Название |
|------|-----|----------|
| Фон | `#1C0F08` | Espresso |
| Панель | `#2A1508` | Dark roast |
| Сетка | `#4A2E1A` | Medium roast |
| Текст | `#F5E6D3` | Cream |
| Акцент 1 | `#D4874E` | Caramel |
| Акцент 2 | `#F2C882` | Honey latte |
