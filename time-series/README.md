# ☕ Coffee Vending Machine | Time Series Analysis

Анализ временных рядов и прогнозирование выручки кофейного автомата на основе транзакций **2024–2025 года**. Ноутбук строит два независимых прогноза (SARIMA и Prophet), декомпозирует ряд на тренд и сезонность, и автоматически выявляет аномальные дни.

## Зачем это важно

Знать сколько кофе продаётся сегодня - недостаточно. Время-серийный анализ отвечает на вопросы которые дашборд не закрывает:

- **Куда идёт бизнес?** - STL-декомпозиция отделяет долгосрочный тренд от недельного шума, давая честный ответ: бизнес растёт, стагнирует или падает.
- **Сколько продадим в следующем месяце?** - SARIMA и Prophet дают независимые прогнозы с доверительными интервалами. Согласие двух моделей повышает уверенность в числе; расхождение сигнализирует о неопределённости.
- **Какие дни были нетипичными?** - аномалии ±2σ указывают на поломки, незапланированные акции или внешние события, которые стоит объяснить.
- **Какой напиток растёт быстрее?** - отдельный прогноз по топ-4 напиткам показывает разные траектории спроса и помогает планировать запасы избирательно.
- **Когда именно пик?** - недельная и суточная сезонность даёт расписание нагрузки с точностью до часа, что напрямую влияет на логистику заправки.

## Требования

```bash
pip install -r requirements.txt
```

Ключевые зависимости для этого модуля: `statsmodels>=0.14`, `prophet>=1.1`.

> **Важно:** если получаешь `ImportError: cannot import name '_lazywhere'` - обнови statsmodels:
> ```bash
> pip install -U statsmodels
> ```
> Это конфликт версий между старым `statsmodels` и новым `scipy`.
